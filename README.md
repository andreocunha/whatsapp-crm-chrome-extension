# whatsapp-crm-chrome-extension
Extensão para navegador que transforma o whatsapp em um kanban (crm)

Para a extensão funcionar corretamente, primeiro você deve instalar ela como desenvolvedor e depois atualizar (F5) na página do whatsapp para funcionar.
Ela pega todos os contatos salvos no indexDB do navegador, então provavelmente vai aparecer muito mais contato do que tem na lista de conversas na lateral padrão do whatsapp.
As fotos do constatos são restauradas do db local tbm, então se você passar por todos os contatos e depois atualizar (F5) o site elas vão aparecer no kanban também.
